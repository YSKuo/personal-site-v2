## 關於目前的狀態

現在在 [Lidemy 程式導師實驗計畫第四期](https://github.com/Lidemy/mentor-program-4th) 的最後階段，一邊跟課一邊進行 Final Project 的前置作業，說是前置作業其實也就是處理設計的部分，因為自己對設計有興趣先前也在 BenQ 當過設計實習生，理所當然想利用這個 project 累積更多經驗，目前 User Flow 快處理完了，順利的話下週應該可以開始畫 Wireframe 吧。

但除了跟課還有 Final Project 以外，自己也想建立個人網站所以就順手弄了。

## 關於這個網站

### 製作
這個網站是用 [Gatsby](https://www.gatsbyjs.com/) 和它的框架 [Reflex](https://reflexjs.org/) 做的，剛開始學網頁設計就一直很喜歡做個人網站，大約一年前學完 HTML 和 CSS 就做了一個 [個人網站](https://yskuo.github.io/CV/) 出來，這第一個網站基本上都是手刻，當時不會 JavaScript 所以胡亂用了一些 jQuery 的套件，不知道那些代碼的意義但看到效果有出來就非常滿意了，而現在更懶了所以這個網站是用了一堆套件弄出來的（笑。

### 想法
原本有在 [CoderBridge](https://www.coderbridge.com/series/e6ca520d20de4f8497e17d1a8d95245f) 和 [Medium](https://medium.com/@crowley3141) 寫文章，但現在太久沒更新基本荒廢了。之後可能會搬文章過來這邊，所以這文章雖是開站首 po，但若有日期比這篇還早的文章，那就應該是別的地方搬來的。

而之後應該不會在 CoderBridge 那繼續發文了，至於在 Medium 那邊則可能會同步發表一些和程式碼比較不相關的文。

這個網站的其他部分應該都是用英文敘述，大概只有 blog 的文章會有中文，或許之後也會發表英文文章練練英文寫作。

此外，還有很多功能沒加到這個網站，總之慢慢加進來。
